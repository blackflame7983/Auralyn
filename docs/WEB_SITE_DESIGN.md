## 1. プロジェクト概要

### 1.1 コンセプト
「**Intuitive VST Hosting**」
プロ品質の音声処理を、誰でも直感的に扱えるようにする。「難しい設定」や「複雑な配線」からの解放を訴求する。

### 1.2 ターゲット
- **OBS初心者**: 配信の音質を上げたいが、設定に悩みたくない層。
- **歌枠・音楽鑑賞勢**: 音質（ASIO/ビットパーフェクト）にこだわる層。
- **Discord利用者**: 通話相手にクリアな声を届けたい層。

### 1.3 テクニカル要件
- **HTML5 / CSS** (Tailwind CSS 推奨, Dark Mode Only)
- **レスポンシブ対応**: Mobile First
- **言語**: 日本語のみ
- **[UPDATE] 対応OS**: Windows 11 (64-bit) ※Windows 10はサポート対象外
