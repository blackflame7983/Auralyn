# かんたんセットアップウィザード：目的別動作マニュアル

Auralynの「かんたんセットアップ」では、最初に選択した利用目的（Goal）に応じて、診断内容と自動設定される出力先が変化します。

## 選択肢ごとの動作の違い

### 1. 🎥 OBSで配信 (OBS Streaming)
配信ソフト「OBS Studio」などに音声送るための設定です。

*   **前提条件**: 仮想オーディオケーブル（VB-CABLE等）のインストールが**必須**です。
*   **動作**:
    *   **判定**: 仮想ケーブルがインストールされているか厳密にチェックします。
    *   **エラー**: 仮想ケーブルが見つからない場合、「出力デバイスが見つかりません」というエラーを表示し、セットアップを中断します。
    *   **自動設定 (推奨)**:
        *   **入力 (Input)**: お使いのマイク
        *   **出力 (Output)**: `CABLE Input` (VB-Audio Virtual Cable)

### 2. 🎧 そのほか通話 (Discord / Voice Chat)
DiscordやZoomなどの通話アプリでエフェクトボイスを使用するための設定です。

*   **前提条件**: 仮想オーディオケーブル（VB-CABLE等）のインストールが**必須**です。
*   **動作**:
    *   **判定**: 仮想ケーブルがインストールされているか厳密にチェックします。
    *   **エラー**: 仮想ケーブルが見つからない場合、「出力デバイスが見つかりません」というエラーを表示します。
    *   **自動設定 (推奨)**:
        *   **入力 (Input)**: お使いのマイク
        *   **出力 (Output)**: `CABLE Input` (VB-Audio Virtual Cable)

### 3. 🔧 とりあえず試す (Just Testing)
自分の声をヘッドホンで聴きながら、エフェクトの効果を試したり録音を行ったりするための設定です。

*   **前提条件**: 仮想ケーブルは**不要**です。
*   **動作**:
    *   **判定**: 仮想ケーブルがなくてもエラーになりません。
    *   **自動設定 (推奨)**:
        *   **入力 (Input)**: お使いのマイク
        *   **出力 (Output)**: 現在使用中のスピーカー/ヘッドホン (モニター出力)
    *   **注意**: このモードのまま通話アプリや配信ソフトを使うと、相手にエフェクト音が届かない（または音が重複する）場合があります。

---

## トラブルシューティング

**Q. 「出力デバイスが見つかりません」と表示される**
A. 「OBSで配信」または「そのほか通話」を選んだ場合、音の受け渡しのために「VB-CABLE」などの仮想オーディオデバイスが必要です。画面の指示に従ってインストールしてください。

**Q. 自分の声が聞こえない（モニタリングしたい）**
A. 配信・通話モードでは、ハウリング防止のため基本的に自分の声はミュート（モニターオフ）になるよう設計される場合があります（※今後の実装による）。モニターしたい場合は、「とりあえず試す」モードを使うか、設定から出力先をヘッドホンに変更してください。
