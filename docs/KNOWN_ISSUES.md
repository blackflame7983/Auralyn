# 既知の課題と改善案

## セットアップウィザード
- **ロジックの単純化**: 現在のウィザードのロジックは非常にシンプルであり、ユーザーの複雑なセットアップや特定のデバイス構成を十分にカバーできていない可能性があります。動的な分析というよりは、静的なルーティングに近い動作です。
- **ウィザード内でのデバイス切り替え不可**: ウィザードのフロー内でオーディオデバイスを切り替えたり再選択したりすることができません。デバイス設定を変更するには、ウィザードを終了してメインアプリケーションの設定画面から行う必要があります。

## 機能と表現のギャップ
- **「自動設定」の表現**: ランディングページのコピーは、「自動ミックス」機能を連想させる過剰な約束を避けるために修正されました（例：「自動で作ります」→「スタート地点を用意します」）。将来的には、初期のビジョンにより近づけるために、実際のオートゲインやスマートルート機能を実装することが望まれます。
